## Project Description
- 개인의 포트폴리오를 구성할 수 있는 포트폴리오 사이트입니다.
- 간단한 소개와 SNS 링크, 경력과 학력, 수상과 자격증, 기술 스택, 진행한 프로젝트 경험 등을 기술할 수 있습니다.
<br>

## Project Stack
- **Backend** : Kotlin
- **Frontend** : Thymeleaf, JavaScript
- **Framework** : SpringBoot
- **Database**: MySQL
